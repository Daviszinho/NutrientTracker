# **App Name**: Nutrient Tracker

## Core Features:

- Data Input: Allow users to mark the number of portions for each nutrient category (Agua, Azucar, Carbohidratos, Proteina, Fruta, Grasa, and Vegetales) for each day of the week with an 'X'.
- Weekly Table Display: Display the nutrient data in a table format, with columns representing days of the week and rows representing nutrient categories.
- Portion Limit Input: Allow users to input the maximum number of portions for each nutrient category in the row header.
- Session Storage: Save the marked portion information for the duration of the user's session.

## Style Guidelines:

- Primary color: Soft green (#90EE90) to represent health and nutrition.
- Background color: Light grey (#F0F0F0) for a clean and neutral backdrop.
- Accent color: Light yellow (#FCE883) to highlight totals.
- Body and headline font: 'PT Sans' for a modern, readable style.
- Use a clear, tabular layout with distinct columns and rows for easy data entry and viewing.
- Simple icons for each nutrient category to aid visual recognition.
- Subtle transitions and highlights on data input to provide feedback.
